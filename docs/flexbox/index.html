FlexBox

Alinhando elementos em uma dimensão, sem sofrimento

Role para baixo para começar

O que é FlexBox?

Até agora, alinhar caixas na tela era um pesadelo. O Flexbox resolve isso: ele organiza os elementos ao longo de um eixo (uma linha OU uma coluna).

Tudo começa aplicando display: flex no elemento pai (o container). Os filhos viram automaticamente itens flexíveis.

  • Eixo principal: a direção em que os itens se alinham.
  • Eixo cruzado: o eixo perpendicular a ele.
.container {
    display: flex;
}

/* Os filhos abaixo viram
   "itens flex" automaticamente */
<div class="container">
    <div>A</div>
    <div>B</div>
    <div>C</div>
</div>

Container: direção e quebra

Estas são as propriedades longas (longhand) que controlam o fluxo dos itens:

flex-direction

Define o eixo principal:

  • row (padrão) — esquerda p/ direita
  • row-reverse — direita p/ esquerda
  • column — de cima p/ baixo
  • column-reverse — de baixo p/ cima

flex-wrap

Define se os itens podem "quebrar" para a próxima linha:

  • nowrap (padrão) — tudo em uma linha só
  • wrap — quebra para a linha de baixo
  • wrap-reverse — quebra para cima

Atalho: flex-flow

Em vez de escrever flex-direction e flex-wrap separados, dá para juntar os dois no atalho flex-flow:

/* Forma longa (longhand) */
.container {
    flex-direction: row;
    flex-wrap: wrap;
}

/* Forma curta (shorthand) — mesma coisa! */
.container {
    flex-flow: row wrap;
}

Recomendação do W3C: a especificação de CSS do W3C encoraja o uso dos atalhos (como flex-flow). Além de deixar o código mais enxuto, o shorthand reseta automaticamente os valores que você não informou — evitando bugs em que uma propriedade antiga "sobra" e atrapalha o layout.

Alinhando os itens

Aqui mora a mágica do Flexbox — alinhar tudo com uma linha de código:

justify-content

Alinha no eixo principal:

  • flex-start / flex-end
  • center
  • space-between
  • space-around
  • space-evenly

gap

Espaço entre os itens: gap: 20px;

align-items

Alinha no eixo cruzado:

  • stretch (padrão)
  • flex-start / flex-end
  • center
  • baseline

align-content

Alinha várias linhas no eixo cruzado (só faz efeito com wrap).

💡 Combo campeão para centralizar QUALQUER coisa: display: flex; justify-content: center; align-items: center;

Propriedades dos Itens

As propriedades acima vão no container. Estas vão em cada item filho:

  • order: muda a ordem visual (padrão 0).
  • flex-grow: quanto o item cresce para ocupar espaço sobrando (padrão 0).
  • flex-shrink: quanto o item encolhe quando falta espaço (padrão 1).
  • flex-basis: o tamanho base do item antes de crescer/encolher.
  • align-self: alinha só aquele item no eixo cruzado (sobrescreve o align-items).
/* Forma longa (longhand) */
.item {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 200px;
}

.item-especial {
    order: -1;      /* vai pro início */
    align-self: center;
}

Atalho: flex

Os três "flex-" do item (grow, shrink, basis) também têm um atalho — o flex:

/* Forma longa */
.item {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 0;
}

/* Forma curta — a ordem é: grow shrink basis */
.item {
    flex: 1 1 0;
}

/* Atalhos famosos que você vai ver por aí: */
.item { flex: 1; }      /* cresce e divide o espaço igualmente */
.item { flex: none; }   /* tamanho fixo, não cresce nem encolhe */

Recomendação do W3C: a especificação de Flexbox diz explicitamente que autores devem preferir o atalho flex em vez das propriedades longas separadas, justamente porque ele reseta corretamente os valores omitidos e evita comportamentos inesperados.

Mão na Massa

Exercício 1: O container flex

Crie uma <div> com três caixinhas dentro. Aplique display: flex no container e observe como as três, que antes ficavam empilhadas, agora ficam lado a lado.

Exercício 2: Centralização perfeita

Faça uma caixa ocupar a tela inteira (height: 100vh) e centralize um texto bem no meio dela (horizontal e vertical) usando apenas justify-content e align-items.

Exercício 3: Menu de navegação

Crie uma barra (menu) com o logo à esquerda e três links à direita, na mesma linha. Dica: use display: flex e justify-content: space-between. Adicione um gap entre os links.

Exercício 4: Cards responsivos com shorthands

Crie um container com 4 cards. Use flex-flow: row wrap no container (atalho!) para que os cards quebrem de linha quando a tela ficar pequena. Em cada card, use o atalho flex: 1 1 250px para que eles cresçam e dividam o espaço, mas nunca fiquem menores que 250px. Redimensione a janela e veja a mágica acontecer.
