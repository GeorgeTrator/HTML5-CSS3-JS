Grid Layout & Media Queries

Layouts em duas dimensões e sites que se adaptam a qualquer tela

Role para baixo para começar

Flexbox vs Grid

O Flexbox organiza itens em uma dimensão (linha OU coluna).

O Grid organiza em duas dimensões ao mesmo tempo (linhas E colunas) — perfeito para o layout geral da página: cabeçalho, menu lateral, conteúdo, rodapé.

Começa com display: grid no container.

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}
/* Cria 3 colunas de tamanho igual.
   Os filhos preenchem célula a célula
   automaticamente. */

Definindo a Grade

grid-template-columns

Define as colunas.

grid-template-rows

Define as linhas.

A unidade fr (fração) distribui o espaço disponível proporcionalmente.

.container {
    display: grid;

    /* 3 colunas: 1 fração cada */
    grid-template-columns: 1fr 1fr 1fr;

    /* repeat() evita repetição */
    grid-template-columns: repeat(3, 1fr);

    /* minmax(): nunca menor que 150px,
       nunca maior que 1 fração */
    grid-template-columns:
        repeat(3, minmax(150px, 1fr));
}

🔑 repeat(3, 1fr) é o mesmo que 1fr 1fr 1fr, só que mais limpo. E minmax() é o segredo da responsividade em grids.

Espaçamento: gap

Assim como no Flexbox, o gap controla o espaço entre as células. Ele também tem forma longa e atalho:

/* Forma longa (longhand) */
.container {
    row-gap: 20px;      /* espaço entre linhas */
    column-gap: 40px;   /* espaço entre colunas */
}

/* Forma curta (shorthand) — ordem: linha coluna */
.container {
    gap: 20px 40px;
}

/* Um valor só = mesmo espaço nos dois sentidos */
.container {
    gap: 20px;
}

Posicionando os Itens

Cada item pode ocupar mais de uma célula. As propriedades longas:

  • grid-column-start
  • grid-column-end
  • grid-row-start
  • grid-row-end

As linhas da grade são numeradas a partir de 1. Um item vai de uma linha inicial até uma final.

/* Forma longa (longhand) */
.destaque {
    grid-column-start: 1;
    grid-column-end: 3;   /* ocupa 2 colunas */
    grid-row-start: 1;
    grid-row-end: 2;
}

Atalhos: grid-column / grid-row / grid-area

/* grid-column: início / fim  (junta start e end) */
.destaque { grid-column: 1 / 3; }

/* grid-row: início / fim */
.destaque { grid-row: 1 / 2; }

/* span = "ocupe N colunas a partir daqui" */
.destaque { grid-column: 1 / span 2; }

/* grid-area junta TUDO: row-start / col-start / row-end / col-end */
.destaque { grid-area: 1 / 1 / 2 / 3; }

Recomendação do W3C: a especificação de CSS do W3C encoraja os atalhos como grid-column, grid-row e grid-area. Eles resetam os valores omitidos e deixam o código muito mais legível do que as quatro propriedades longas separadas.

Layout por Nomes

O jeito mais visual de montar uma página: dar nomes às áreas com grid-template-areas e posicionar cada item com grid-area:

.pagina {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-areas:
        "cabecalho cabecalho"
        "menu      conteudo"
        "rodape    rodape";
}

.cabecalho { grid-area: cabecalho; }
.menu      { grid-area: menu; }
.conteudo  { grid-area: conteudo; }
.rodape    { grid-area: rodape; }

Repare como o próprio CSS já "desenha" o formato da página. Lindo, né?

Alinhamento no Grid

O Grid alinha tanto os itens dentro das células quanto a grade inteira. Cada par tem seu atalho:

Itens nas células

  • justify-items (horizontal)
  • align-items (vertical)
  • Atalho: place-items
/* longhand */
justify-items: center;
align-items: center;

/* shorthand */
place-items: center;

A grade inteira

  • justify-content / align-content
  • Atalho: place-content

Um item só

  • justify-self / align-self
  • Atalho: place-self

Media Queries

De nada adianta um layout lindo no computador se ele quebra no celular. As Media Queries aplicam CSS só quando certas condições são verdadeiras — geralmente, o tamanho da tela.

  • max-width: "até no máximo X" (telas pequenas).
  • min-width: "a partir de X" (telas grandes).
/* Estilo padrão (qualquer tela) */
.container {
    grid-template-columns: repeat(3, 1fr);
}

/* Em telas de até 768px (tablet/celular):
   vira uma coluna só */
@media (max-width: 768px) {
    .container {
        grid-template-columns: 1fr;
    }
}

Mobile First

A abordagem recomendada é Mobile First: escreva o estilo base pensando no celular e use min-width para adicionar complexidade conforme a tela cresce.

/* Base: celular (uma coluna) */
.galeria { grid-template-columns: 1fr; }

/* Tablet: 2 colunas a partir de 600px */
@media (min-width: 600px) {
    .galeria { grid-template-columns: repeat(2, 1fr); }
}

/* Desktop: 4 colunas a partir de 1024px */
@media (min-width: 1024px) {
    .galeria { grid-template-columns: repeat(4, 1fr); }
}

📱 Breakpoints comuns: ~600px (celular grande), ~768px (tablet), ~1024px (desktop). Também dá para testar @media (orientation: landscape) para telas deitadas.
