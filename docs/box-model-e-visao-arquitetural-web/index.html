Tudo em HTML5 é uma caixa!

📦 Padding - Espaço Interno

padding-top: Espaço acima do conteúdo

padding-right: Espaço à direita

padding-bottom: Espaço abaixo

padding-left: Espaço à esquerda

Exemplo:

padding-top: 20px;
padding-right: 30px;
padding-bottom: 20px;
padding-left: 30px;

🌍 Margin - Espaço Externo

margin-top: Espaço acima da caixa

margin-right: Espaço à direita

margin-bottom: Espaço abaixo

margin-left: Espaço à esquerda

Exemplo:

margin-top: 30px;
margin-right: 0px;
margin-bottom: 30px;
margin-left: 0px;

🎨 Border - Borda da Caixa

border-top: Borda acima

border-right: Borda à direita

border-bottom: Borda abaixo

border-left: Borda à esquerda

Exemplo (largura | estilo | cor):

border-top: 4px solid #3498db;
border-right: 4px solid #3498db;
border-bottom: 4px solid #3498db;
border-left: 4px solid #3498db;

⭐ Outline - Linha Externa

outline-width: Espessura da linha

outline-style: Estilo (solid, dashed, dotted, etc)

outline-color: Cor da linha

outline-offset: Distância da borda

Exemplo:

outline-width: 2px;
outline-style: dashed;
outline-color: #e74c3c;
outline-offset: 5px;

⚡ Importância Semântica

Para que serve cada propriedade?

📦 Padding

Função: Cria espaço interno ao redor do conteúdo

Quando usar: Para dar "respiração" ao texto/conteúdo dentro da caixa

Exemplo: Um botão com padding oferece espaço confortável para o clique

⚡ Importância Semântica

Para que serve cada propriedade?

🌍 Margin

Função: Cria espaço externo entre caixas

Quando usar: Para separar elementos na página, controlar o layout

Exemplo: Margin entre parágrafos evita que o texto fique muito junto

⚡ Importância Semântica

Para que serve cada propriedade?

🎨 Border

Função: Define uma borda visível ao redor da caixa

Quando usar: Para destacar, separar visualmente ou criar contornos

Exemplo: Um card com border cria uma "moldura" visual clara

⚡ Importância Semântica

Para que serve cada propriedade?

⭐ Outline

Função: Cria uma linha fora da borda (não ocupa espaço no layout)

Quando usar: Para acessibilidade - indicar foco em navegação por teclado

Exemplo: Quando você pressiona TAB em um input, o outline marca qual elemento tem foco

✨ Importante para inclusão digital!

Resumo do Box Model

📦 Padding: Espaço interno ao redor do conteúdo

🌍 Margin: Espaço externo entre caixas

🎨 Border: Borda visível ao redor da caixa

⭐ Outline: Linha externa para foco/acessibilidade

📝 Exercício 007: O Mistério do Transbordamento

1. Crie um arquivo HTML e defina uma div com o id secao (ela será o nosso container pai).

2. Dentro dela, crie outra div com o id caixa e coloque um texto dentro dela.

3. No CSS, dê 100% de largura para a #caixa e depois adicione 30px de padding e 10px de borda.

4. O Desafio: Use o box-sizing: border-box; para impedir que a caixa azul escape do limite cinza do pai!

<!-- Se liga nesses novos conceitos:
(1) "id" é como o "CPF" de uma tag. Cada tag pode ter UM e SOMENTE UM id.
    O id permite que você aplique estilos próprios à tag e somente àquela tag!
(2) "div" é uma tag NÃO SEMÂNTICA. Ela é usada para criar blocos e 
    ajudar na estilização do layout.
-->

<div id="secao">
    <div id="caixa">Sou uma caixa de texto!</div>
</div>

Antes de aplicar box-sizing: border-box

Após aplicar box-sizing: border-box;

🎯 Display: inline-block

O melhor dos dois mundos!

Combina:

  • ✅ Fica na mesma linha (como inline)
  • ✅ Respeita padding, margin e border (como block)
  • ✅ Respeita width e height (como block)

Código:

display: inline-block;

Caso de uso: Criar botões lado a lado, cards em linha, menus horizontais

Na prática: Display

📝 Exercício 008: O Menu de Botões

1. Crie uma div com o id menu.

2. Dentro dela, coloque três tags de link <a> (elas são inline por padrão).

3. No CSS, tente dar width: 150px e height: 50px para os links. O que acontece?

4. Agora, mude o display para block e depois para inline-block. Qual deles resolve o menu?

<!-- Estrutura do Menu -->
<div id="menu">
    <a href="#">Home</a>
    <a href="#">Produtos</a>
    <a href="#">Contato</a>
</div>

Obs: Tags <a> são "inline" e ignoram largura/altura até você mudar o display! Hey! Você sabia que href="#" é uma maneira rápida de referenciar a própria página?

Exemplo Prático

HTML:

<button>Botão 1</button>
<button>Botão 2</button>
<button>Botão 3</button>

CSS:

button {
  display: inline-block;
  padding: 10px 20px;
  margin: 5px;
  background: #3498db;
  color: white;
}

Resultado: Os botões ficam lado a lado com espaçamento!

⚠️ Cuidado com Whitespace!

Com inline-block, espaços em branco no HTML criam pequenos gaps:

❌ ERRADO (cria gaps):

<button>1</button>
<button>2</button>
<button>3</button>

✅ CORRETO (sem gaps):

<button>1</button><button>2</button><button>3</button>

Dica: Usamos Flexbox ou Grid para evitar esse problema! (veremos depois)
