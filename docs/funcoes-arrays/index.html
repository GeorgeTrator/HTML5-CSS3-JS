O que são Funções?

Lembram do def no Python? No JavaScript, usamos a palavra function!

Uma função é como uma receita de bolo: você escreve as instruções uma única vez, guarda com um nome, e pode executar (chamar) quantas vezes quiser.

  • Evita repetição de código.
  • Deixa o código organizado e fácil de ler.
  • Podem receber dados (parâmetros) e devolver resultados (return).
// Declarando a função
function darBoasVindas(nome) {
    alert("Olá, " + nome + "!");
}

// Chamando a função
darBoasVindas("Turma");

Arrays (Listas)

Na aula passada vimos variáveis simples. Mas e se eu quiser guardar o nome de 30 alunos? Criar 30 variáveis seria um pesadelo.

Um Array é uma variável especial que pode guardar múltiplos valores ao mesmo tempo, organizados por posições numéricas (índices), que sempre começam no Zero.

// Criando um Array (Lista)
let jogos = ["Minecraft", "Valorant", "Fifa"];

// Acessando valores
console.log(jogos[0]); // Mostra: Minecraft
console.log(jogos[1]); // Mostra: Valorant

// Adicionando um novo item no final
jogos.push("Roblox");

Unindo Arrays e Laços (For)

A verdadeira mágica acontece quando juntamos os Arrays com a estrutura de repetição for que aprendemos na última aula.

Podemos usar a propriedade .length (tamanho) para que o laço percorra a lista inteira automaticamente!

let frutas = ["Maçã", "Banana", "Laranja", "Uva"];

// frutas.length vale 4. O "i" vai de 0 até 3.
for (let i = 0; i < frutas.length; i++) {
    console.log("Eu gosto de " + frutas[i]);
}
